# ngn-presentation

## Get started
```sh
npm install
npm run rlmd:dev
```

Or:
```sh
pnpm install
pnpm rlmd:dev
```

This installs reveal-md and serves slides (`slides.md` file) at http://localhost:1948. It also watches for changes to the file due to `-w` flag.

`reveal-md` code: https://github.com/webpro/reveal-md
